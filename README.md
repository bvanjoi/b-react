# A React template for my self

Components:

- typescript
- eslint, prettier
- babel
- jest,
- husky: lint-staged, commitlint, changelog
- webpack, webpack-dev-server, portfinder
  - css: sass, postcss: autoprefixer

## Usage

clone this repo:

```sh
git clone https://github.com/bvanjoi/b-react.git
```

```sh
yarn
```

Then

```sh
yarn dev
```
